# HackYeah2024

## Instalacja

Szegóły instalacji qiskit: https://docs.quantum.ibm.com/guides/install-qiskit

#### Create Virtual Env:


```bash
python3 -m venv ./.venv
```

#### Activate it:

```bash
source ./.venv/bin/activate
```

#### Install requirements:

```bash
pip install -r requirements.txt
```
